## 简介 · About

**Hivora** 是一款面向严肃交易者的桌面级复盘工具，提供历史行情多周期回放、结构化复盘笔记、双盲模拟推演与交易统计。数据本地优先存储，支持云端多设备同步。

本仓库用于发布 Hivora 客户端的安装包，**请前往 [Releases](https://github.com/HangRuan/Hivora/releases/latest) 下载最新版本**。

> Hivora is a desktop chart-replay and review tool for serious traders — multi-timeframe historical playback, structured review journaling, double-blind simulation, and trade statistics. This repository hosts the official release downloads.

## 🚀 安装 · Installation

### macOS

1. 下载并双击 `.dmg`，将 **HivoraData** 拖入「应用程序 / Applications」文件夹。
2. 首次打开若提示「无法打开，因为无法验证开发者」，请前往
   **系统设置 → 隐私与安全性**，在底部找到关于 HivoraData 的提示，点击 **「仍要打开 / Open Anyway」**，并在弹窗中再次确认即可。

   > （macOS System Settings → Privacy & Security → Open Anyway）

### Windows

1. 下载并解压 `HivoraData-windows-x86_64.zip`。
2. 进入解压后的文件夹，双击 **`HivoraData.exe`** 即可运行。
3. 若 SmartScreen 提示「未知发布者」，点击 **更多信息 → 仍要运行**。

> 首次启动会自动解压内置运行时（用于在隔离沙盒中运行数据插件），请稍候片刻。

## ✨ 核心功能 · Features

- **历史行情回放** — 按标的 / 日期检索，从任意历史点位逐根推进。
- **多周期联动** — 分钟 / 小时 / 日线同步回放，宏微观结构对比。
- **结构化复盘笔记** — 笔记与图表切片精准锚定；买卖时机、止盈止损结果、方向判断多维记录。
- **交易统计** — 自动汇总胜率、盈亏比、平均盈亏等指标。
- **双盲模拟推演** — 遮蔽未来走势进行虚拟买卖，剔除后见之明，客观还原决策水平。
- **插件与指标市场** — 数据源插件在隔离沙盒中运行，自定义指标按需扩展。
- **跨平台 + 云端同步** — macOS / Windows 原生客户端，本地存储 + 多设备同步，多语言界面。
